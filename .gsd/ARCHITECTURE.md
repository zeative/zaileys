## Overview

Zaileys is a lightweight and feature-rich TypeScript wrapper around the internal `baileys` library, tailored for building flexible and robust WhatsApp automation bots and applications. It provides high-level APIs for authentication, database synchronization, media manipulation (via FFmpeg/Jimp), and listening to various WhatsApp signal events (messages, groups, community features, call privacy).

## Components

### Client Core
- **Purpose:** Entry point for creating the WA socket, registering components, injections, and managing initialization of cleanup/health routines.
- **Location:** `src/Classes/client.ts`
- **Dependencies:** `baileys`, `jetdb`, `zod`, `Config/*`, `Auth/*`

### Authentication & DB Layer
- **Purpose:** Responsible for persisting Baileys auth credentials, caching group metadata, and persisting chat stores across restarts via single-file JSON DB (`JetDB`).
- **Location:** `src/Config/database.ts`, `src/Auth/`
- **Dependencies:** `jetdb`, `baileys`

### Media Library & Core Tools
- **Purpose:** Manipulating audio, images, documents, and stickers, integrating with ffmpeg system installers. Managing internal proxy state injections.
- **Location:** `src/Library/`
- **Dependencies:** `@ffmpeg-installer/ffmpeg`, `fluent-ffmpeg`, `jimp`

### Signal Listeners
- **Purpose:** Handling WhatsApp network packets (messages, read receipts, call statuses, group updates) into standardized event emit objects inside `store.events`.
- **Location:** `src/Signal/`, `src/Listener/`
- **Dependencies:** `Client (core)`

## Data Flow

1. Entry point initializes the `Client`, loading local `JetDB` JSON cache for credentials mapping them to standard `baileys` AuthState.
2. The network connection establishes, registering various system plugins and middleware (if available under options).
3. The `Listener/` catches packet signals (`baileys` socket event emitter), enriches context via `getRoomName`, limits data with `rate-limiter-flexible`, and emits formatted Zailey events.
4. User logic attached via `client.on(...)` executes payload logic.
